### 선택정렬
- 데이터 무작위로 여러 개 있을 때, 
- **가장 작은 데이터 선택해 맨 앞에 있는 데이터와 바꾸기**
- 그 다음 작은 데이터를 선택해 앞에서 두 번째 데이터와 바꾸기
- 예
  - 데이터의 개수 N개 
  - 가장 작은 데이터를 앞으로 보내는 과정 N-1 반복해 정렬 완성 

### 선택정렬 시간 복잡도
- N-1번 만큼 가작 작은 수 찾아서 맨 앞으로 보내기
- 매번 가장 작은 수 찾기 위한 비교 연산이 필요
- 2중 반복문 사용
- N + (N - 1) + (N - 2) + ... + 2 = N X (N + 1) / 2 = (N^2 + N) / 2
- O(N^2)
