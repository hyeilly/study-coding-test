- 빅-오 표기법 O(n)을 기준으로 수행시간 계산 
- 연산횟수 계산 = 알고리즘 시간 복잡도 n값에 데이터 최대 크기 대입하여 도출

  - 예외 처리
- 오류 찾기
  1. 변수 초기화 오류 찾아보기

  3. 잘못된 변수 사용 오류 찾아보기
